MANUAL DE INSTRUÇÕES DO SISTEMA

Posicione o mouse sobre o Frame e clique na tela abaixo para criação da Figura.

OBS:
1)O sistema não aceita teclas maiúsculas
2)O triângulo ser clicado um pouco deslocado do centro para esquerda e para baixo. 
3) A Figura criada onde o mouse estiver no Frame
4) Para funcionar o resize, isto é, o redimesionamento das Figuras o mouse deve ser clidado na Figura.
5) As figuras também pode ser criadas com os botões a esquerda do Frama 



Para criar uma elipse - tecle "e"

Para criar um tiângulo da - tecle "t"

Para criar uma retangulo - tecle "r"

Para criar um círculo da - tecle "c"

Para criar um quadrado - tecle "q"

Para criar um texto - tecle "x"



Para deletar a última figura - tecle a tecla "Delete"
Para arrastar uma figura clique sobre esta
Para aumentar o tamanho clique na tecla  "PgUp" (seta para cima)
Para diminuir o tamanho clique na tecla "PgDn" (seta para baixo)
